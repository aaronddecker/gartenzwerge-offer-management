# 20.06.2026

Heute wurde die rollenbasierte Frontend-Navigation im Gartenzwerge-Projekt erweitert.
Neben der bestehenden Authentifizierung wurden Admin-only Bereiche jetzt nicht nur in
der UI ausgeblendet, sondern auch auf Routenebene geschützt.

Employees können /analytics und /offered-services nicht mehr direkt über die URL öffnen,
sondern werden zurück zum Dashboard geleitet. Admins können diese Bereiche weiterhin
aufrufen. Außerdem wurde die Dokumentation für den v0.10.0-Meilenstein aktualisiert.

Wichtig gelernt:
Rollenlogik besteht aus mehreren Ebenen. Das Frontend verbessert die Nutzerführung,
aber die echte Berechtigungsprüfung bleibt Aufgabe des Backends.
