# 12.06.2026 - Gartenzwerge Order Management

Heute wurde der Meilenstein v0.7.0 Authentication Foundation begonnen.
Ich habe verstanden, warum ASP.NET Core Identity eine produktionsnähere Grundlage ist als eine komplett eigene User-Entity.
Die Infrastructure-Schicht wurde für Identity vorbereitet: ApplicationUser wurde erstellt, AppDbContext auf IdentityDbContext umgestellt, Identity-Pakete wurden versionstechnisch korrigiert, eine Migration wurde erstellt und auf PostgreSQL angewendet.
Außerdem wurde Identity über AddIdentityCore in der DependencyInjection registriert.
Build und Tests waren grün, anschließend wurde der technische Identity-Infrastructure-Setup-Commit erstellt.
