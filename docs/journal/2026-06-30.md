# 2026-06-30 – Releases v0.14.0 and v0.15.0

## Completed

- Released v0.14.0 with editable order details, order filters and status badges
- Released v0.15.0 with a live dashboard and an analytics page
- Added revenue, conversion and a 12-month trend chart on analytics
- Built the chart and the rolling numbers as DIY components, no libraries
- Updated the docs and synced develop with main

## Learnings

- A milestone can be frontend-only when the backend endpoint already exists.
- Sharing business rules keeps the dashboard counts and the filters in sync.
- Small details matter: reduced motion, touch instead of hover, pixel-snapped digits.
- A complete milestone earns a release, an in-between step is just a pull request.
