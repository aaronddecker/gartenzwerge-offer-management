# Gartenzwerge Außenservice – Angebots- und Auftragsmanagement

Eine Full-Stack-Anwendung zur Verwaltung von Kunden, Leistungen, Angeboten und Aufträgen für einen Garten- und Außenservice.

## Ziel

Das Projekt demonstriert moderne Full-Stack-Entwicklung mit C#, ASP.NET Core, PostgreSQL, React und Docker.
